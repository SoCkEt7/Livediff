# Livediff marketing control center

This folder contains copy-ready, rule-aware assets for promoting the public Livediff repository without spam or platform-rule violations.

## Assets

- `audience-and-positioning.md` — audience, message, channel fit.
- `devto-article.md` — ready-to-edit dev.to / Hashnode article draft.
- `reddit-playbook.md` — subreddit targets, posting rules, tailored post drafts.
- `github-growth-plan.md` — README/About/topics/release/discussion plan.
- `post-bank.md` — short posts, comments, replies, CTAs.
- `launch-calendar.md` — 14-day sequence.

## Non-negotiables

- Do not ask for votes, stars, or artificial engagement.
- Do not mass-post the same text across communities.
- Read each community rule page before posting.
- Use one post per community with a useful angle specific to that community.
- Reply to every substantive comment with technical detail.
- Never mention content generation tooling in public marketing copy.
